Module track

Utility for tracking collections of entities.

Functions

track.set (coll, entity, data) Set tracking on an entity.
track.remove (coll, entity) Stop tracking an entity.
track.get (coll, entity) Get the set of entities in a collection, or get the data associated with a specific entity.
track.each (coll, f) Do something for each tracked entity in a collection.
track.any (coll, f) Check whether any entity in a collection matches a predicate.


Functions

track.set (coll, entity, data)
Set tracking on an entity.

Parameters:

  • coll The collection to track the entity in.
  • entity The entity to track.
  • data Optional; data to store along with this tracking entry.
track.remove (coll, entity)
Stop tracking an entity.

Parameters:

  • coll The collection to stop tracking the entity in.
  • entity The entity to stop tracking.
track.get (coll, entity)
Get the set of entities in a collection, or get the data associated with a specific entity.

Parameters:

  • coll The collection to read.
  • entity The entity to read data for, or omitted/nil to get the set of entities in the collection.

Returns:

    The data for the entity if passed, otherwise the collection of entities and mapping of data for this collection.
track.each (coll, f)
Do something for each tracked entity in a collection.

Parameters:

  • coll The collection to iterate.
  • f The function to call for each entity. Will be called with (entity, data).
track.any (coll, f)
Check whether any entity in a collection matches a predicate.

Parameters:

  • coll The collection to check.
  • f The predicate to check for each entity.

Returns:

    The return value of the first invocation of f that returned a non-false non-nil value, if present; otherwise nil.
